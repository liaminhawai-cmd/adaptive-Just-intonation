Lattice

A ratio-based just-intonation instrument with Ben Johnston notation. Colour = consonance with what's playing.

Reference

Add a note

Right-click any node (lattice or sequencer), drag up/down to lock onto a just interval, scroll to set the prime limit, click to drop.

Notes

JohnstonratiocentsHzHD♪

♪ = in the current key signature: only those pitch classes get sequencer lanes (each in every octave). Comma-pump freely on the lattice, keep the grid to ~7 notes, re-key as you go.

MIDI Output (→ FL Studio)

Not connected. Route through a virtual MIDI cable (loopMIDI on Windows, IAC Driver on Mac) into FL Studio. Mono mode bends one channel per note — plays one line at a time correctly on any synth. Set your synth's pitch-bend range to match the field above (2 is the default).
